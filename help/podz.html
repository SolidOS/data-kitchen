Podz

Dual-pane file browser for Solid pods. Pick a pod in each panel's dropdown, log in, then drag/drop files between sides.

Per-pod settings

Each pod has its own ⚙ gear (top-right of the panel) for per-pane display preferences:

  • Hide dot-files / # / ~ — filter the listing.

Editor keybindings (Default / Emacs / Vim) live on the editor itself when you open a file for editing.

How to customize the Solid Data Kitchen

Copy installFolder/config.default.json to installFolder/config.json. Then edit the following fields:

Window size and position

Position can be either "center" or an integer (no quotes).

Start Page

Leave as an empty sting to start with the Data Kitchen intro page or set it to any URI.

Bookmarks

Add as many as you wish. The URIs you supply will be opened in the dataBrowser.

File Root

This is the document root of the local fileSystem, similar to an HTTPD document root - if fileRoot is "/home/me/", then a URL of file:///public/ will be mapped onto the folder at /home/me/public/. The default fileRoot is the myPod folder of the installation. You can change it to any folder on your filesystem, including, if you wish and are able, the root folder "/".

The fileRoot is not only the root folder for browsing, it is where Data Kitchen will look for your profile and preferences exactly as it does on a pod. So you should recursively copy all of the files and folders in the myPod folder to your new fileRoot. So if you set the fileRoot to /home/me/, copy ${installation folder}/myPod/* to /home/me/.

How to customize the Solid Data Kitchen

Copy installFolder/config.default.json to installFolder/config.json. Then edit the following fields:

Window size and position

Position can be either "center" or an integer (no quotes).

Start Page

Leave as an empty sting to start with the Data Kitchen intro page or set it to any URI.

LOCAL_BASE

This variable allows you to use "./" as a shortcut in the dataBrowser. If LOCAL_BASE is set to "/home/me/", a request in the dataBrowser URI bar for ./foo.txt will find the file "file:///home/me/foo.txt". You could have gotten to the same place by using that full URI. If this variable is missing or empty, it will default to the "myPod" folder of your installation folder.

REMOTE_BASE

This variable allows you to use "/" as a shortcut in the dataBrowser. If REMOTE_BASE is set to "https://me.example.org/public/", a request in the dataBrowser URI bar for "/foo.txt" will look for the resource at "https://me.example.org/public/foo.txt". You could have gotten to the same place by using that full URI. If this variable is missing or empty, you will not be able to use this shortcut.

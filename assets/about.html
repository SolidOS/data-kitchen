Solid Data Kitchen

A stand-alone data browser for everyone and everything

This project provides a kitchen where the user is cook, and the power user is chef. Live your life in a read-write world of linked data!

Use all your solid pods, including your local files as your zeroth pod. Bring in all the stuff in your life shared by friends and colleagues. Bring in public data like dbpedia and wikidata and the LOD cloud.

How to cook

  • DataBrowser - explore pods and other linked data sources
    • Query button - use current DataBrowser location as a SPARQL endpoint
    • Save button - save documents from the DataBrowser to a local file or to a pod
    • URI shortcuts - quick links to pods, files, and ontologies
  • File menu - copy/move files in a pod, between pods; between pods and local file system
  • Query form - search multiple sources at once, results link back to DataBrowser
  • Bookmarks menu - comes pre-loaded with samples, see below on how to customize
  • See also : customizin the kitchen | URI shortcuts | setting up a local profile

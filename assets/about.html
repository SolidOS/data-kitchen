Solid Data Kitchen

A stand-alone data browser for everyone and everything

This project provides a kitchen where the user is cook, and the power user is chef. Live your life in a read-write world of linked data!

Use all your solid pods, including your local files as your zeroth pod. Bring in all the stuff in your life shared by friends and colleagues. Bring in public data like dbpedia and wikidata and the LOD cloud. Check out the bookmarks menu which has samples of several of these uses.

How to cook

  • DataBrowser - explore pods and other linked data sources
  • File menu - copy/move files in a pod, between pods; between pods and local file system.
  • Query - search multiple sources at once - Wikipedia, Internet Archive, etc.
  • Bookmarks menu - visit your favorite pods.
  • Help menu - learn about Solid and about customizing the Data Kitchen.

Want to save some keystrokes? Read the help menu about URI shortcuts.

Podz

File browser for Solid pods. Pick a pod in the dropdown and log in. Click a file — or an item's ⚙ — to open it in the panel on the right (view, edit, rename, delete, permissions…); drag the bar between the panels to resize them.

The ◫◫ button (bottom-left) switches to the dual-pane view: two pod browsers side by side, for drag/drop copying files between pods. There, opening an item uses a dialog instead.

Per-pod settings

Each pod has its own ⚙ gear (top-right of the panel) for per-pane display preferences:

  • Hide dot-files / # / ~ — filter the listing.

Editor keybindings (Default / Emacs / Vim) live on the editor itself when you open a file for editing.
